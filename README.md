# Wander India AI Based Travel Planner

## Developed by
Naga Pradeep

## Project Description
This is an AI-based travel planner that helps users generate travel itineraries for different cities in India. It provides smart suggestions based on user preferences.

##  Features
- AI-generated travel plans
- Multiple city support
- Dynamic itinerary generation
- User-friendly interface

##  Technologies Used
- Frontend: React / HTML / CSS
- Backend: Node.js / Python
- APIs: OpenAI / Travel APIs

##  How to Run
1. Install dependencies  
2. Run backend  
3. Run frontend  
4. Open in browser  

##  Project Structure
- frontend  
- backend  
- assets  
- scripts
